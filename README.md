# To-Do App

## Project Overview
This to-do app is a user-friendly task management tool designed to help users organize their daily activities. Built with HTML, CSS, and JavaScript, it features a clean, modern design and a responsive layout, making it accessible on various devices.

## Features
- **Task Addition**: Users can add new tasks to their to-do list using a simple input form.
- **Dynamic Task Display**: Tasks are dynamically displayed in an organized list.
- **Task Completion and Deletion**: Users can mark tasks as completed or delete them entirely.

## Technologies Used
- **HTML**: For structure.
- **CSS**: Styling with Google Fonts (Poppins) and a linear gradient background.
- **JavaScript**: For interactive functionalities like adding, completing, and deleting tasks.

## Installation and Setup
No installation is required. The app can be directly accessed through a web browser.

## Usage
Simply type a task into the input field and press enter to add it to the list. Tasks can be marked as complete or deleted using the buttons next to each task.
